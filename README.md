# Sistema de Gerenciamento de Biblioteca em C

Este projeto consiste em um sistema de gerenciamento de biblioteca desenvolvido na linguagem **C**, com foco na prática de estruturas de dados, manipulação de arquivos e programação estruturada.

O sistema permite realizar o cadastro, consulta, empréstimo, devolução e listagem de livros por meio de um menu interativo executado no terminal. Além disso, os dados são armazenados em um arquivo `.txt`, garantindo a persistência das informações entre diferentes execuções do programa.

## Funcionalidades

* Cadastro de novos livros;
* Consulta de livros por código;
* Empréstimo de livros;
* Devolução de livros;
* Listagem completa do acervo;
* Validação para impedir códigos duplicados;
* Controle de disponibilidade dos livros (Disponível/Emprestado);
* Salvamento automático dos dados em arquivo de texto;
* Carregamento automático do acervo ao iniciar o sistema.

## Tecnologias utilizadas

* Linguagem C
* Programação estruturada
* Structs
* Vetores
* Manipulação de arquivos (`FILE`)
* Manipulação de strings (`string.h`)
* Estruturas de repetição e seleção
* Funções

## Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de consolidar conhecimentos fundamentais em programação na linguagem C, aplicando conceitos como modularização do código, organização de dados com estruturas (`struct`), persistência de informações utilizando arquivos e implementação de um sistema CRUD em ambiente de console.

É um excelente exercício para compreender como aplicações de gerenciamento funcionam internamente antes da utilização de bancos de dados e interfaces gráficas.
